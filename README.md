# restaurant-page

This is a project to practice DOM manipulation for the Odin Project course.

I was able to get some experience using ES6 modules and bundling with Webpack.
This 'restaurant site' has buttons to switch to the other pages of the site.
I was able to practice using different modules to export functions to create these
pages when the buttons are clicked.

Here is a link to the project lesson that I was following:
https://www.theodinproject.com/lessons/node-path-javascript-restaurant-page
